# Café Friends Website

This is a static responsive website developed for the fictional coffee shop “Café Friends” in Riga as part of a Study Paper in Information Technology at EKA University of Applied Sciences.

## Features
- Fully responsive design (mobile-first)
- Built with HTML5 and CSS3 only
- Sections: Home, Menu, About Us, Gallery, Contact
- Semantic HTML and accessibility best practices
- Local SEO elements (structured data, meta tags)

Open `index.html` in any browser to view the site locally.
